Shiori adds bookmarks to a PDF file which has no bookmarks.


## Install

```bash
python3 -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Run 
python3 shiori.py

and follow the GUI screen indications.


## Environment
Python 3.8 or lator
